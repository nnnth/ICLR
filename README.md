    numpy pandas matplotlib sklearn xgboost

### Usage

For symptom predict
```shell
python symptom.py
```

For disease predict
```shell
python disease_train.py
```

The model will be saved in a file

For test and submit

```
python test.py
```

Then, the file "ans.pk" can be submmited.
